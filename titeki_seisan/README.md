# YAGNI 原則とは何か答えよ。

You Aren’t  Gonna Need It. の略。必要になるまで機能を追加していけないという意味。理由は以下の通り。
- 重要なのは「今どうあるべきかなの」であるのに、将来のことを考えるのはミッションから気をそらしている。  
- 時間は貴重である。  
- もし必要なくなった場合、それに費やした時間が無駄になる。

# ソースコードを段階的に読む方法を答えよ.

- 内部構造を説明したドキュメントがあればそれを読む。  
- ディレクトリ構造を分析する  
- ファイル構成を分析する  
- 略語を調査する  
- 関数の呼び出し関係を確認する  
- 関数の内容を確認する
