## 学びを効率化するためのステップ
1. 技術の全体像を把握する
2. スコープを決める
3. 成功の基準、ゴールを決める
4. 資料をできる限り多く集める
5. 学習プランを立てる
6. 資料をフィルタリングする
7. 使い始められるようにする方法を学ぶ
8. 使って遊ぶ
9. 役に立つことができるところまで学ぶ
10. 教える
